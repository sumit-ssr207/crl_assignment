APIs


Base URL

The BASE URL is the common url used as prefix for all API call. [Base URL] : http://127.0.0.1:8000

Get userlist with pagination

This call can be used to get userlist in pagination manner GET : /user_listing/all_user_list_api
Input Parameter Type Description
page int Page Number. If not provided, default will be 1
records int Number of records per page. If not provided, default will be 1000
Sample Response

Search users by input parameters

This call can be used to get matching users based on input parameters. GET : /user_listing/get_filtered_data_api
Input Parameter Type Description
first_name String First Name
last_name String Last Name
username String Userid
gender String Gender
Sample Response

Note

APIs have been secured with Django authentication. If it needs to be accessed from third party apps then JWT token based mechanism can be implemented.

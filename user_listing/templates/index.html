Assignment Statement


USER LISTING :
Develop an application in Python using Django Framework which will consume
https://randomuser.me APIs and store data in the database. Once you store data in the database,
expose APIâ€™s where you are able to fetch a list via pagination for the list of users available.

GOALS
1. Initiate a Project
2. Setting up an API which will get User Details from the API and store in the database.
Preferably relational database.
3. Store the details in database
4. Implement and expose an API which will get the data in a paginated manner
5. Implement and expose an api which search users by name / id / gender in database
6. Once you click on a user detail, show the entire information related to that user in an
inner navigation
7. Please note that the pagination / search & filter api needs to be done in your project
instead of the randomuser api.
8. Bonus : Additional consideration for optimizing for Performance for a large set of records.

SPECIFICATIONS
1. Complete this assignment in 1 week
2. You can refer to https://randomuser.me/documentation for documentation.
3. Use the git version control. Once your assignment is completed please share the
standard zip / tar of your source code including the all the git meta.
4. You can use any JavaScript framework to complete the front end related tasks.
5. Your assignment should also include the README.md with instructions to install and setup
DB if any
